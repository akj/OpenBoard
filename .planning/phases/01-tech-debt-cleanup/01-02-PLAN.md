## `_resolve_move_context` Helper — Locked Constraints (Codex MEDIUM)

**Return type:** `typing.NamedTuple` (NOT a `@dataclass`) per Codex MEDIUM. NamedTuple gives:
- Immutability by construction (no field assignment).
- Zero ceremony (no `__init__`, no `field()`, no `default_factory`).
- Familiar tuple-unpacking idiom for callers that prefer it: `difficulty, fen, book = self._resolve_move_context()`.
- Per CLAUDE.local.md: prefer simplicity; the NamedTuple has 3 named fields, well within NamedTuple's ergonomic sweet spot.

**Signature:**
```python
from typing import NamedTuple

class MoveContext(NamedTuple):
    """Resolved context for the next computer move — output of Game._resolve_move_context().

    Pure data; no side effects in construction. Callers consume the fields explicitly
    or unpack as a tuple.
    """
    difficulty_config: DifficultyConfig
    fen_before: str
    book_move: chess.Move | None
```

**Purity contract (Codex MEDIUM — baked into acceptance_criteria):**

`_resolve_move_context()` MUST:
1. Validate `self.config.mode` against the set of computer-move modes (raise `GameModeError` if invalid).
2. Resolve `difficulty_config` from `self.config.difficulty` via the existing lookup logic.
3. Capture `fen_before = self.board_state.board.fen()` BEFORE any move is made.
4. Optionally consult `self.opening_book` and return a `book_move` if one is available; else `None`.

`_resolve_move_context()` MUST NOT:
- Call `.send()` on any signal (no `self.computer_move_ready.send(...)`, no `self.move_made.send(...)`).
- Call `wx.CallAfter(...)` or any wx-thread-marshalling primitive.
- Push a move onto `self.board_state` (no `make_move`, no `_board.push`).
- Trigger callbacks (no calling `engine.get_best_move_async(callback=...)`).
- Mutate `self` beyond reading state.

These constraints are auditable by greppable patterns and a test:

```python
def test_resolve_move_context_is_pure(self):
    """Verifies TD-07 / Codex MEDIUM: _resolve_move_context has no side effects."""
    import inspect
    source = inspect.getsource(Game._resolve_move_context)
    assert ".send(" not in source, "TD-07: _resolve_move_context must not emit signals"
    assert "wx.CallAfter" not in source, "TD-07: _resolve_move_context must not marshal to wx thread"
    assert "make_move(" not in source, "TD-07: _resolve_move_context must not push moves"
    assert "get_best_move_async" not in source, "TD-07: _resolve_move_context must not call engine"
    assert "board_state._board.push" not in source, "TD-07: _resolve_move_context must not push directly"
```
